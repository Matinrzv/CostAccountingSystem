# 🍫 سیستم محاسبه قیمت تمام‌شده - شرکت شادی شکلات

این نرم‌افزار دسکتاپ به سفارش **شرکت تولیدی شادی شکلات** طراحی و پیاده‌سازی شده است. هدف اصلی این سیستم، محاسبه دقیق بهای تمام‌شده هر محصول (نان بستنی قیفی، حصیری و برشی) بر اساس مواد اولیه مصرفی و هزینه‌های سربار (دستمزد، برق، بسته‌بندی، اجاره و ...) در بازه‌های زمانی مشخص می‌باشد.
